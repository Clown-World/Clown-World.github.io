git AAA

date: 2017-03-01 19:19:34

Git工作的一般流程：

1.获得一个Git管理的工作区
2.添加修改推送
3.分支
4.拉取别人的提交
5.处理冲突

Git的基本命令

1.git init 新建Git代码库
2.git clone 下载一个项目和它的整个代码历史
3.git config –list 显示当前的Git配置
4.git config[–global] user.name “[name]” 提交代码时的用户名
5.git config[–global] user.email “[email]” 提交代码时的邮箱
6.git add [file1] 指定文件到暂存区
7.git add 添加当前目录的所有文件到暂存区
8.git commit -m [message] 提交暂存区到仓库区
9.git commit -a 提交工作区自上次commit之后的变化，直接到仓库区
10.git branch 列出所有本地分支
11.git branch -r 列出所有远程分支
12.git checkout 切换到上一个分支
13.git branch -d 删除分支
